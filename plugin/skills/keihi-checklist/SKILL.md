# keihi-checklist

## Role / Inputs / Success Criteria

| 項目 | 内容 |
|------|------|
| **役割** | 業種別・会社情報をもとにした月次経理チェックリストを生成 |
| **入力** | Memory内の会社プロフィール or ユーザーの直接説明 |
| **成功条件** | 1人で30分以内に完結する手順書 + 各ステップにClaudeプロンプト付き |

## When to Use

- 「月次経理のチェックリストを作って」
- 「来月から自分で経理できる手順が欲しい」
- 「私の業種でよくある見落とし経費も含めて」

## ワークフロー

### Step 1: 会社情報の確認

Memory または直接入力から以下を取得する:
- 業種
- 会社名
- よく使う取引先
- 毎月かならず発生する支出パターン

不足していれば簡潔に聞く（最大2問）。

### Step 2: チェックリスト生成

以下の構成で作成する。

---

**テンプレート構成:**

```markdown
# 月次経理チェックリスト — [会社名] ([業種])

## 毎月の作業手順（所要時間: 約30分）

### □ Step 1: 通帳・カード明細を取り込む（5分）
銀行・クレジットカードの明細CSVをダウンロードして保存する。

**Claudeへのプロンプト:**
「この明細データは私の会社の○月分の出入金です。
すべての行の仕訳を判断して、判断できないものは確認してください。
freeeに接続できていれば、確認後に全件登録してください。」

---

### □ Step 2: 止まった取引先に答える（5〜10分）
Claudeが確認してきた取引先に、自社の使い方を答える。

**Claudeへのプロンプト（テンプレ）:**
「〇〇は[使い方]なので、[勘定科目]にしてください。全件の仕訳を完了してください。」

---

### □ Step 3: freee登録を確認する（5分）
freeeにログインして、今月登録された取引を確認する。

**Claudeへのプロンプト:**
「今freeeに登録したデータを確認してください。登録漏れや分類ミスがあれば教えてください。」

---

### □ Step 4: 見落とし経費の確認（5分）
業種特有の経費が抜けていないかチェックする。

[業種別の見落としやすい経費をここに挿入]

---

### □ Step 5: 月次レポートの確認（5分）
費目別の内訳と前月比較を確認する。

**Claudeへのプロンプト:**
「今月のfreeeデータを分析してください。費目の内訳と、来月削れそうな支出を教えてください。」

---

## ✅ 完了チェック
- [ ] 全件の仕訳が完了した
- [ ] freeeに登録された
- [ ] 見落とし経費を確認した
- [ ] 月次レポートを確認した
```

---

### Step 3: 業種別の「見落とし経費」を追加

業種に応じて以下を差し込む:

| 業種 | よくある見落とし経費 |
|---|---|
| 小売・卸売 | 仕入先への交通費 / 展示会出展費 / 荷造り梱包費 |
| 飲食 | 食材サンプル費 / 衛生用品 / 店舗消耗品 |
| 建設・工務店 | 現場交通費 / 工具消耗品 / 安全用品 |
| IT・制作 | サブスクリプション更新 / 参考書籍 / クラウドサービス |
| 士業・コンサル | 専門誌・書籍 / 交流会費 / 名刺印刷 |
| 個人事業主全般 | 自宅兼事務所の家賃按分 / 通信費按分 / 健康診断費 |
